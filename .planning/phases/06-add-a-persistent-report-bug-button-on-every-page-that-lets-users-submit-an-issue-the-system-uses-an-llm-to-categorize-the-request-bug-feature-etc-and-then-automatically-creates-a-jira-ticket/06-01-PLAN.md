    - ParseResult type: { success: true; category: string; ticketKey: string; ticketUrl: string } | { success: false; error: string }
    - Valid JSON string: returns { success: true, category, ticketKey, ticketUrl }
    - JSON wrapped in markdown code fences (```json\n...\n```): strips fences, parses, returns success shape

    - Unparseable input (prose, empty string): returns { success: false, error: string }
    - Parsed object missing required fields: returns { success: false, error: string }
